# Node REST API

A REST API written in Node.js to learn about using Node with no frameworks. Based off Brad Traversy's [Vanilla Node.js REST API project](https://youtu.be/_1xa8Bsho6A).

#### Routes

```
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id
```
